# URLShortener
A URL shortener in python with Flet. 

# How to run:
Create a virtual environment:

```commandline
python -m venv venv 
```

Activate the virtual environment:

```commandline
source venv/bin/activate 
```

Install the dependencies: 
```commandline
pip install -r requirements.txt
```

Run the project: 
```commandline
flet app.py -d -w
```
